# 🛡️ 兒童臉部隱私保護工具

自動偵測照片中的兒童臉部並用可愛 Emoji 遮蓋，保護兒童隱私。

🌐 **線上使用**: https://cagoooo.github.io/child-face-privacy/

---

## ✨ 功能特色

| 功能 | 說明 |
|------|------|
| 🔍 自動偵測 | 使用 face-api.js 偵測臉部並判斷年齡 |
| 😊 Emoji 遮蓋 | 6 種可愛表情可選擇 |
| ✏️ 手動編輯 | 新增/移除/調整遮蓋 |
| 🔄 對比滑桿 | 拖曳比較原圖與處理後 |
| 📦 批次處理 | 一次上傳多張照片 |
| 💾 下載功能 | 單張 PNG 或批次 ZIP |
| 📱 PWA 支援 | 可安裝為 App、離線使用 |
| 🎨 RWD 設計 | 支援手機、平板、桌面 |

## 📋 版本紀錄

### v1.0.0 (2023-12-23)

- ✅ RWD 響應式設計

## 🔧 可優化改良項目

### 高優先

| 項目 | 說明 |
|------|------|
| 🔍 **更強的偵測模型** | 可改用 SSD MobileNet 或 MTCNN 以提高偵測準確度 |
| 🎭 **馬賽克選項** | 除了 Emoji，可添加馬賽克/模糊效果 |
| 📐 **臉部旋轉偵測** | 目前 Emoji 不會隨臉部角度旋轉 |

### 中優先

| 項目 | 說明 |
|------|------|
| 📊 **處理進度條** | 每張照片顯示即時處理進度 |
| 🔒 **隱私模式** | 本地處理，確保照片不上傳伺服器 (已實現) |
| 🖼️ **圖片壓縮** | 上傳前壓縮大型照片以加速處理 |
| 📏 **自訂 Emoji 大小** | 每個遮蓋可獨立調整大小 |

### 低優先

| 項目 | 說明 |
|------|------|
| 🌙 **亮色主題** | 目前只有深色主題 |
| 🌐 **多語言支援** | 英文/日文/韓文 |
| 📷 **相機拍照** | 直接拍照處理 |
| ↩️ **撤銷/重做** | 編輯操作的撤銷功能 |
| 🎨 **自訂 Emoji** | 上傳自己的貼圖 |
| 💾 **匯出設定** | 儲存常用設定 |

### 技術優化

| 項目 | 說明 |
|------|------|
| ⚡ **Web Worker** | 將臉部偵測移到背景執行緒 |
| 📦 **程式碼分割** | 延遲載入非必要模組 |
| 🔄 **模型快取** | IndexedDB 快取偵測模型 |
| 📱 **觸控手勢** | 雙指縮放、旋轉遮蓋 |

## 🛠️ 技術架構

- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **臉部偵測**: face-api.js (TinyFaceDetector)
- **打包下載**: JSZip + FileSaver.js
- **PWA**: Service Worker (Network-First)
- **部署**: GitHub Pages
